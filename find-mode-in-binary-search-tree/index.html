501.二叉搜索树中的众数

发表于 2020-09-24 分类于 leetcode 阅读次数：

1. 中序遍历+辅助数组

中序遍历获得有相同值的BST的非严格递增序列，存入辅助数组中，然后统计众数。
时间O(n)，空间O(n)。

2. 中序遍历

在中序遍历的过程中，可以直接记录和统计当前数据的出现次数，跟之前出现过的最大次数比较即可，不需要辅助数组。
时间O(n)，空间O(n)。
