155. 最小栈

发表于 2020-05-12 更新于 2021-03-09 分类于 leetcode 阅读次数：

1. 直接模拟

用一个变量记录最小元素的位置就可以了。

2. 辅助栈

用一个辅助栈记录过程中的最小值，用额外O(n)的空间将pop()的复杂度降低为O(1)。
以下两种方式都可以：

  • 同步辅助栈
    数据栈：3, 1, 2, 4, 5, 1
    辅助栈：3, 1, 1, 1, 1, 1
  • 不同步辅助栈
    数据栈：3, 1, 2, 4, 5, 1
    辅助栈：3, 1, 1
